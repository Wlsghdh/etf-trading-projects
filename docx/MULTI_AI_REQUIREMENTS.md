## 1. 멀티AI 3x3 그리드 리뉴얼

현재: ML Prediction / Fundamental / Market 행 × GPT / Gemini / Claude 열
변경: **Technical / Fundamental / Market** 행 × GPT / Gemini / Claude 열

### 1-1. Technical (기존 "ML Prediction" → "Technical" 으로 변경)

**AI에게 요청할 내용:**
- 선택한 종목의 **기술적 분석** (RSI, MACD, 볼린저밴드, 이평선 등)
- **실적 그래프**: 최근 분기별 EPS/매출 추이
- 금융사(증권사)에서 나오는 **종목 리포트 종합 분석**
  - 목표가, 투자의견(매수/중립/매도), 컨센서스

**함께 보여줄 시각 요소:**
- [ ] 실적 차트 (분기별 EPS/매출 바 차트)
- [ ] 기술지표 요약 대시보드 (RSI 게이지, MACD 히스토그램 등)
- [ ] 증권사 컨센서스 테이블 (목표가 min/max/avg)

### 1-2. Fundamental (기존 유지, 내용 강화)

**AI에게 요청할 내용:**
- **DART 정기공시 / 분기보고서** 기반 분석
  - 해외 주식이므로 → SEC EDGAR (10-K, 10-Q) 또는 Yahoo Finance 재무제표
- PER, PBR, ROE, 부채비율 등 밸류에이션 지표
- 동종업계 대비 비교 분석

**함께 보여줄 시각 요소:**
- [ ] 재무제표 요약 테이블 (매출/영업이익/순이익 3개년)
- [ ] 밸류에이션 비교 차트 (동종업종 PER 비교 등)
- [ ] 배당 이력 차트 (있는 경우)

### 1-3. Market (기존 유지, 데이터 대폭 강화)

**AI에게 요청할 내용:**
- 현재 시장 환경이 해당 종목에 미치는 영향 종합 분석
- 뉴스 반영 (최근 주요 뉴스 + 시장 심리)
- 거시경제 데이터 해석

**함께 보여줄 시각 요소 (핵심 — "한번에 정보 다보기"):**
- [ ] **환율 그래프** (USD/KRW 추이)
- [ ] **공포지수** (VIX 또는 CNN Fear & Greed Index)
- [ ] **금 가격** 차트
- [ ] **금리** (미국 기준금리 / 10년물 국채 수익률)
- [ ] **연준(Fed) 기준금리** 변동 타임라인
- [ ] 시장 지표 대시보드 (S&P500, 나스닥, 다우 일일 변동)

### 1-4. 공통 UX

- 각 셀(GPT/Gemini/Claude)의 텍스트 응답은 기존대로 Markdown 렌더링
- **각 행(Technical/Fundamental/Market) 상단에 공통 데이터 패널** 추가
  - 그래프/차트/지표를 한눈에 보고 → 그 아래에 3개 AI 텍스트 분석
  - 또는 행 좌측에 차트 패널 + 우측에 3열 AI 텍스트
- 종목 선택 후 "분석" 버튼 1번이면 3행 × 3열 전부 동시에 호출

## 2. 매매전략 페이지 (신규 사이드바 메뉴)

**위치**: 사이드바에 "매매전략" 메뉴 추가 (`/strategy`)

### 2-1. 매매 복기 (Review)

- 과거 매수/매도 내역을 시간순으로 조회
- 각 거래에 대해:
  - 매수 시점의 종목 스냅샷 (당시 가격, 기술지표, 시장 상황)
  - 매도 시점의 스냅샷
  - 수익/손실 결과
- 필터: 종목별, 기간별, 수익/손실별

### 2-2. 시나리오 예측 (핵심 기능)

**플로우:**
1. 종목 선택 (예: AAPL)
2. 기준 날짜 선택 (예: 2026-03-01)
3. "예측" 버튼 클릭
4. → **5가지 시나리오 차트** 동시 출력:

| 시나리오 | 설명 | 차트 색상 |
|----------|------|-----------|
| 급등 (Bull+) | 강한 상승 모멘텀 | 진한 초록 |
| 상승 (Bull) | 점진적 상승 | 연한 초록 |
| 보합 (Neutral) | 횡보 / 박스권 | 회색/노랑 |
| 하락 (Bear) | 점진적 하락 | 연한 빨강 |
| 급락 (Bear+) | 강한 하락 / 패닉셀 | 진한 빨강 |

- 각 시나리오는 기준 날짜부터 N일(예: 63일) 후까지의 **예상 가격 경로를 캔들차트로 시각화**
- 5개 시나리오를 하나의 차트에 오버레이 또는 5분할 패널로 표시
- 각 시나리오별 **예상 수익률**, **최대 낙폭(MDD)**, **도달 확률** 표시

### 2-3. 가상 매매 전략 시뮬레이션

- 사용자가 **가상 매수/매도 주문**을 입력
  - "AAPL $150에 100주 매수 → $170에 50주 매도 → $180에 나머지 매도"
- 해당 전략의 **백테스트 결과** 시뮬레이션
  - 진입/청산 시점을 차트에 마킹
  - 총 수익률, 보유 기간, 승률 등

## 3. 데이터 소스 정리

| 데이터 | 소스 | 용도 |
|--------|------|------|
| 기술지표 (RSI, MACD 등) | etf2_db (이미 보유) | Technical 행 |
| 실적 (EPS/매출) | Yahoo Finance API 또는 AI 프롬프트 | Technical 행 |
| 증권사 리포트 | AI 프롬프트 (웹 검색 기반) | Technical 행 |
| 재무제표 (10-K/10-Q) | SEC EDGAR / Yahoo Finance | Fundamental 행 |
| 환율 (USD/KRW) | open.er-api.com (이미 사용 중) | Market 패널 |
| VIX (공포지수) | Yahoo Finance (`^VIX`) | Market 패널 |
| 금 가격 | Yahoo Finance (`GC=F`) 또는 TradingView | Market 패널 |
| 금리/국채 | FRED API (이미 키 보유: `9caba366...`) | Market 패널 |
| 뉴스 | 기존 뉴스 API (`/api/news`) | Market 행 AI 프롬프트 |
| 과거 매매 내역 | trading.db (이미 보유) | 매매전략 복기 |
| 시나리오 예측 | ML 모델 + Monte Carlo / AI | 매매전략 시나리오 |

## 4. 구현 우선순위 (제안)

### Phase 1: 멀티AI 그리드 리뉴얼 (즉시)
1. "ML Prediction" → "Technical" 라벨 변경
2. AI 프롬프트 재작성 (Technical / Fundamental / Market 각각)
3. Market 행 상단에 시장 데이터 패널 추가 (환율/VIX/금리/금)

### Phase 2: 매매전략 페이지 기본 (다음)
4. `/strategy` 페이지 + 사이드바 메뉴 추가
5. 시나리오 예측: 종목+날짜 선택 → 5개 시나리오 차트
6. 매매 복기: trading.db 기반 과거 주문 조회

### Phase 3: 고도화 (이후)
7. Market 패널에 실시간 지표 위젯 추가
8. 재무제표 자동 크롤링 (SEC EDGAR / Yahoo Finance)
9. 가상 매매 시뮬레이션 (백테스트)
10. 증권사 리포트 통합
